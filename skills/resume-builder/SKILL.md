# 简历构建器 (Resume Builder)

这个技能指导如何通过对话方式，为用户创建一份**视觉精美、排版专业**的 A4 纸 HTML 简历。

## 核心理念

- 简历是个人品牌的第一印象，需要独特且有记忆点
- 排版要干净利落，便于招聘官快速扫描
- 设计风格遵循内置的美学标准：大胆、有辨识度、拒绝通用审美。详见 `references/design-guidelines.md`
- **目标输出**：一个独立的 HTML 文件，内嵌 CSS/JS，A4 尺寸排版，可直接浏览器打印为 PDF

## 工作流程

### 第一步：信息收集（对话式面试）

与用户进行对话，收集以下信息板块。逐个板块提问，不要一次抛出所有问题。

**基础信息**：
- 姓名、联系方式（手机、邮箱、城市）
- 求职意向（目标岗位、期望城市、薪资范围可选）
- 个人网站/GitHub/LinkedIn（可选）

**教育背景**：
- 学校名称、专业、学历、起止时间
- GPA 或排名（可选）
- 相关课程或荣誉（可选）

**实习/工作经历**（核心板块）：
- 公司名称、岗位、起止时间
- 每个经历提取 2-4 个关键成果（用 STAR 法则：情境-任务-行动-结果）
- 量化数据优先（提升了 X%，完成了 Y 个项目，管理了 Z 人团队）

**项目经验**（核心板块）：
- 项目名称、角色、时间
- 项目简介（1-2 句话）
- 核心技术栈
- 个人贡献和成果（2-3 点）

**技能标签**：
- 编程语言、框架、工具、平台
- 语言能力（中/英/其他）
- 可分级：精通/熟练/了解

**校园经历**（应届生加分）：
- 社团活动、志愿者、竞赛获奖
- 担任角色和主要贡献

**自我评价**（可选）：
- 2-3 句话的自我定位
- 突出个人特色和职业目标

### 第二步：风格确认

在收集完信息后，让用户确认简历风格。提供一个简洁的选择列表：

1. **现代简约** — 黑白为主，大量留白，几何线条，字体优雅
2. **经典商务** — 深蓝或深灰主调，稳重专业，传统排版
3. **创意个性** — 独特配色，不对称布局，展示个性（适合设计/创意岗位）
4. **日式极简** — 极简主义，柔和色彩，留白呼吸感强
5. **科技感** — 深色/渐变，几何元素，适合技术岗位

用户也可以描述自己想要的风格，由技能根据内置的美学指南进行创造。生成前，先阅读 `references/design-guidelines.md` 获取设计方向指导。

### 第三步：生成 HTML 简历

基于收集的信息和选择的风格，生成一个独立的 HTML 文件。

**技术约束**：
- 必须是一个独立的 HTML 文件（inline CSS/JS，无外部依赖）
- 页面尺寸严格为 A4（210mm × 297mm）
- 使用 `@page { size: A4; margin: 0; }` 控制打印区域
- 内容控制在 1-2 页（A4 尺寸下）
- 使用 `@media print` 优化打印效果
- 字体使用系统字体栈（中文: PingFang SC, Microsoft YaHei; 英文: 系统 serif/sans-serif 组合）

**设计约束**：
- 遵循内置美学标准（见 `references/design-guidelines.md`），精心选择字体组合、配色方案、空间排版
- 拒绝使用 Inter、Roboto、Arial 等过度使用的字体
- 配色要有主色+点缀色，避免软弱的均匀调色板
- 关键信息（姓名、公司名、标题）要有层次感
- 使用 CSS 变量统一管理颜色
- 适当的 CSS 动画或微交互是可选的（但要克制）

**A4 布局要点**：
- 内容区域宽度控制在 190mm 以内（留 10mm 页边距）
- 字号不能太小（正文 11px 以上，标题 16px 以上）
- 板块之间有清晰的视觉分隔
- 打印时隐藏所有非必要元素（如按钮、背景色等）

**PDF 导出**：
- 在 HTML 中加入一个「导出 PDF」按钮
- 使用浏览器的 `window.print()` 实现打印为 PDF
- 打印时自动隐藏该按钮
- 添加一段说明文字提示用户：Chrome → 另存为 PDF

### 第四步：交付

生成文件后，告诉用户：
1. 文件保存位置
2. 如何用浏览器打开和打印为 PDF
3. 下一步可以使用 jd-tailorer skill 针对具体 JD 进行定制

## 示例对话

**用户**: 帮我做一份简历
**Claude**: 好的，让我帮你构建一份精美的简历。首先，能告诉我你的姓名和求职意向吗？

**用户**: 我想换一个深色风格的简历
**Claude**: 明白，我会为你设计一个适合技术岗位的深色主题简历。信息我都收集好了，现在开始生成。

## 文件命名

- `resume.html` — 简历文件，保存在用户指定的位置
- 建议用户在项目根目录下保存，如 `resume/index.html`
