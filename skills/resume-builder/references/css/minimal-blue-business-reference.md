# 简约蓝色商务风格参考指南 — Minimal Blue Business Reference

本指南详细剖析了「简约蓝色商务风」的设计原则、框架结构以及实现细节，帮助维护人员及 Agent 更好地理解和产出高质量的简历。

---

## 1. 设计语言与定位

「简约蓝色商务风」旨在在 **稳重可信** 的商务调性与 **现代低噪** 的极简美学之间取得平衡。

*   **色调**：以深蓝色（Navy）作为主基调，辅以亮蓝色（Accent）作为点缀，配合大面积干净的白底以及小面积的浅蓝 wash 区域。
*   **线条**：使用精致、克制的细边框和虚线，替代大面积粗重色块。
*   **结构**：左侧具有一根醒目的彩色装饰竖条，作为整个简历的视觉骨架和呼吸锚点。

---

## 2. 核心模块与框架拆解

### 2.1 页面骨架 (Page Layout)
*   **左侧装饰竖条**：在 A4 `.resume` 页面左侧边缘，使用 6px 宽的线性渐变竖条。该竖条能强力锁住读者的视线，并赋予简历一种工艺感与专业度。
*   **左右页边距**：由于左侧加入了竖条，左边距需要进行适当的补偿增加（例如 `padding-left: calc(var(--page-margin) + 8px)`），以保持视觉上的居中感。

### 2.2 顶部 Header
*   **三栏式布局**：
    *   **左栏**（主栏）：大字号姓名（Bold Serif/Sans-serif） + 推荐求职意向。
    *   **中栏**：多行紧凑的联系信息（电话、邮箱、GitHub/LinkedIn）。
    *   **右栏**：精致的免冠照片框（48px × 60px），外框带有细色边，内嵌照片或保持空白占位。
*   **底部边框**：Header 与下方内容使用 2px 的主色实线进行物理分隔，清晰界定首尾。

### 2.3 章节标题 (Section Title)
*   **左侧色块**：每个 `<h2>` 的标题左侧带有 4px 宽、14px 高的 Accent 色块，作为视线聚焦的引导。
*   **右侧延展线**：标题右侧有一条 `flex-grow: 1` 且高度为 1px 的细淡边框线，向右自然延伸，增加横向层次感。

### 2.4 经历与项目卡片 (Experience/Project Card)
*   **左侧边线**：项目和经历块（`.item-block`）的左侧设计有 2px 宽的浅蓝或灰色实线，鼠标悬停时会渐变为 Accent 蓝色。
*   **顶部分隔线**：同章节内的多个项目/经历块之间，采用 `border-top: 1px dashed` 虚线进行物理分割，在保证区分度的同时避免视觉过载。

### 2.5 轻量标签系统 (Tag System)
*   **轻量徽章**：技能项等标签不使用传统的饱满色块，而是使用带有 1px 细边框 + 3px Accent 颜色左边线的小卡片（`.skill-badge`）。这既能突出重点，又具有很好的 ATS 纯文本阅读流友好性。

### 2.6 自我评价 (Summary Block)
*   **浅蓝 Wash 背景**：自我评价采用浅蓝洗色背景（Palette A 中的 `#F4F8FC` 等），并搭配 4px 宽的 Primary 深蓝左边线，整体形成一个高亮包裹块，作为简历的完美收尾。

## 3. 设计要点对照表 (Do's & Don'ts)

| 应当做 (Do's) | 避免做 (Don'ts) |
| :--- | :--- |
| **主色点缀鲜明**：深蓝配高亮冰蓝，形成明确的对比。 | **平铺大色块**：避免在简历主体中使用深蓝色作为大面积背景（科技感除外）。 |
| **虚实结合**：多项目之间用 dashed 虚线，Header 底部用实线。 | **实线过多**：过多的实线会导致页面显得网格化、死板。 |
| **单页强约束**：通过精细调整行高（1.35）和页边距保持在 1 页 A4 内。 | **任意溢出**：溢出几行到第二页是绝对不能接受的。 |
| **中英文混排优雅**：标题采用 Lexend / Outfit 等英文字体，中文使用系统默认无衬线。 | **字体混乱**：不要混合超过 2 组以上的英文字体系列。 |

## 4. 与「经典商务 (Classic Business)」的差异

| 维度 | 经典商务 (Classic Business) | 简约蓝色商务 (Minimal Blue Business) |
| :--- | :--- | :--- |
| **首要字体** | 经典 Playfair Display 衬线标题 | 现代 Lexend / Outfit / Poppins 无衬线标题 |
| **章节分割** | 简单的底部实线 | 左侧 Accent 色块 + 右侧延展线 |
| **卡片设计** | 纯文本从上到下，无左边线 | 左侧带有实线装饰条，悬停变色，多卡片间使用虚线 |
| **左侧边缘** | 纯白边缘，普通页边距 | 具有 6px 宽的线性渐变彩色装饰竖条 |
| **收尾板块** | 纯文本 | 浅蓝色 Wash 背景的高亮 Summary 块 |
