# 简历设计美学指南

指导简历 HTML 的视觉设计方向。CSS 变量和排版参数在 `css/common.md` 和 `css/<style>.md` 中。

## 设计思维

生成前确定方向：
- **目的**：HR/TL/主管在 6-15 秒内快速扫描
- **基调**：极简克制 / 经典商务 / 创意个性 / 日式留白 / 科技感 — 选一个并极致执行
- **差异化**：拒绝通用 AI 审美，让简历在视觉上有记忆点

## 字体排版

- 拒绝 Inter、Roboto、Arial、Space Grotesk 等过度使用的字体
- 中文字体：PingFang SC（现代）、Noto Serif SC（优雅）、Microsoft YaHei（稳重）
- 标题/正文形成对比层次：字号层级 姓名 > 板块标题 > 公司/学校名 > 正文 > 辅助信息
- 具体搭配见 `color-palettes.md` 字体速查表

## 色彩

- CSS 变量统一管理，主色 + 锐利点缀色 > 软弱均匀的调色板
- 避免紫色渐变配白底等过度套路
- 具体色板见 `css/<style>.md`（每个风格 3 套完整变量）

## 空间构成

- 单页强约束：所有紧凑排版参数在 `css/common.md` 中统一定义
- 双栏布局优先（左侧窄栏技能/联系，右侧主栏经历/项目）
- `orphans: 2; widows: 2;` 避免孤行，但不过度使用 `break-inside: avoid`

## 背景与视觉细节

- 创造氛围，非默认纯白。细线几何、微妙边框、色块分区均可
- 打印时 `@media print` 适当简化但不丢失结构

## 重要原则

实现复杂度匹配美学愿景：极繁需精细代码，极简需克制精准。优雅 = 方向执行到位，非堆砌效果。
