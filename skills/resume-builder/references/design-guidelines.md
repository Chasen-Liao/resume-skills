# 简历设计美学指南

这份指南提炼自前端设计的最佳实践，用于指导简历 HTML 的视觉设计。

## 设计思维

在编写代码之前，先理解上下文并确定一个**大胆的美学方向**：
- **目的**：简历要解决什么问题？谁在看？（HR、技术负责人、部门主管）
- **基调**：选择一个极致的方向——极简克制、现代几何、经典商务、创意个性、日式留白、科技感等
- **约束**：A4 纸打印、严格单页（应届生/初级绝对要求）、中文环境、无外部依赖
- **差异化**：什么让这份简历让人过目不忘？

**关键**：选择一个清晰的美学方向并精准执行。极繁主义的精细和极简主义的克制同样有效——核心是意向性，而非强度。

## 字体排版

- 选择有特色、有品味的字体。拒绝 Arial、Inter、Roboto、Space Grotesk 等被过度使用的字体
- 中文字体推荐：PingFang SC（现代）、Noto Serif SC（优雅）、Microsoft YaHei（稳重）、思源宋体（传统）
- 英文/数字字体推荐：搭配中文字体选择对应的英文衬线/无衬线字体
- 展示性标题配精致正文字体，形成对比层次
- 字号层级：姓名 > 板块标题 > 公司/学校名 > 正文 > 辅助信息
- 具体字体搭配方案参见 `color-palettes.md` 中的字体速查表

## 色彩与主题

- 坚守一个明确的美学调性。用 CSS 变量统一管理颜色
- 主导色 + 锐利点缀色远胜于软弱、均匀分布的调色板
- 避免过度使用的配色：紫色渐变配白色背景
- 常见方向：
  - 黑白 + 单一强调色（经典不失品味）
  - 深色背景 + 高对比文字（科技感）
  - 温暖中性色 + 深绿/深蓝点缀（稳重专业）
  - 柔和米白 + 深棕/深灰（日式极简）
- 具体配色方案参见 `color-palettes.md`：五大风格索引
- CSS 文件在 `css/` 子文件夹，按风格拆分（每个包含 3 套完整色板 + 字体 + 风格 CSS 要点），通用排版在 `css/common.md`

## 空间构成

- 出人意料但不失实用的布局
- 不对称、适度的重叠、对角流动
- 大量留白 OR 高密度控制——二选一，不要折中
- A4 约束下的巧思：利用分栏、边距、间距创造节奏感

**强约束：单页紧凑排版参数**

简历必须控制在 1 页 A4 以内（应届生/初级绝对要求）。以下是确保单页的 CSS 参数规范：

- 页边距：≤ 12mm（`@page { margin: 10mm; }` 或内容区 padding: 8-12mm）
- 正文字号：10-11px，行高 1.3-1.5
- 标题字号：13-15px（板块标题），22-26px（姓名）
- 板块间距：≤ 10mm（约 28-38px at 96dpi）
- 条目间距：≤ 4mm（列表项间 margin-bottom: 2-4px）
- 技能标签：`display: inline` 排列，不换行浪费纵向空间
- 教育背景：可单行紧凑显示（学校 · 专业 · 时间）
- 优先双栏布局：左侧窄栏（30-35%）放技能/联系/教育，右侧主栏放经历/项目
- 使用 `orphans: 2; widows: 2;` 避免内容断裂到下一页
- 避免 `page-break-inside: avoid` 过度使用——会制造不必要的空白

**验证方法**：生成 HTML 后必须用无头浏览器导出 PDF 并检查页数。页数 > 1 则按溢出修复策略调整后重试。

## 背景与视觉细节

- 创造氛围和深度，而非默认纯白背景
- 可在打印友好的前提下使用：细线几何图案、微妙的边框装饰、色块分区、图标点缀
- 打印时通过 `@media print` 适当简化背景，但不丢失结构

## 动画与微交互

- 屏幕浏览时可以有一些克制的小动画（如板块淡入）
- 打印时必须完全移除（`@media print` 中禁用）
- 屏幕效果服务于投递前的视觉体验，打印效果服务于 HR 的实际阅读

## 最重要的原则

实现复杂度要与美学愿景匹配：
- 极繁设计需要精细的代码、丰富的细节、精致的排版
- 极简设计需要克制、精准、对间距和字体吹毛求疵
- 优雅来自于把选定的方向执行到位，而非堆砌效果

**拒绝通用 AI 审美**：不要让每份简历看起来都一样。根据用户的实际背景和目标岗位做出不同的设计选择。
